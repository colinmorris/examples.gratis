### Expand a `.tar.gz` file

<CommandSession>
<Command>ls</Command>
<CommandOutput>fruits.tar.gz</CommandOutput>
<Command>tar -<Flag kw={1}>x</Flag><Flag kw={2}>f</Flag> fruits.tar.gz</Command>
<Command>ls</Command>
<CommandOutput>apple.png banana.png canteloupe.png fruits.tar.gz</CommandOutput>
</CommandSession>

<Gloss>
<Kw kw={1}>Extracts</Kw> the contents of the <Kw kw={2}>file</Kw> <FileRef>fruits.tar.gz</FileRef> into the current directory.
</Gloss>

### Expand a `.tar` file

<Command>tar -xf fruits.tar</Command>

### Compress a directory

<Command>tar -czf fruits.tar.gz fruits/</Command>

<Gloss>
<Kw kw={2}>Compresses</Kw> the directory  <code>fruits/</code> to create the file <code>fruits.tar.gz</code>.
</Gloss>
