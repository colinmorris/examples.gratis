## Create a symlink


<CommandSession>
<Command>ln <Flag
  kw={1} short='s' long='symbolic'
>-s</Flag> ../foo/README.txt <SourceAnnot kw={2}>.</SourceAnnot></Command>
</CommandSession>

Creates a <Kw kw={1}>symbolic</Kw> link in the <Kw kw={2}>current directory</Kw> called `README.txt` which points to the file `../foo/README.txt`.

## Create a symlink with a different name

<CommandSession>
<Command>ln -s ../foo/README.txt readme.md</Command>
</CommandSession>

Creates a symbolic link in the current directory called `readme.md` which points to the file `../foo/README.txt`.

## Symlink a directory

Making a link to a directory uses the same syntax as for a file.

<CommandSession>
<Command>ln -s ../foo/ .</Command>
</CommandSession>

Creates a symbolic link in the current directory called `foo` which points to the directory `../foo`.
