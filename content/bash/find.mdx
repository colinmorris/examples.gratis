## Find by name

<CommandSession>
<Command>find <Kw kw={1}>.</Kw> <Flag kw={2}>-name 'papaya*'</Flag></Command>
<CommandOutput>{`./pictures/papaya.png
./fruit_poetry/papaya.txt
./games/papaya_simulator`}
</CommandOutput>
</CommandSession>

Print the paths of files or directories under <Kw kw={1}>the current directory (`.`)</Kw> whose names <Kw kw={2}>begin with 'papaya'</Kw>.

## `-exec`

The `-exec` option specifies some action to perform on each matching path.

<CommandSession>
<Command>find fruit_poetry -type f -exec wc -l <SourceAnnot
  tt={<p><kbd>{`{}`}</kbd> is a placeholder representing the filename being processed</p>}
  >{'{}'}</SourceAnnot> <SourceAnnot 
  text="`\;` marks the end of the command to perform"
  >\;</SourceAnnot></Command>
<CommandOutput>{`8 fruit_poetry/papaya.txt
6 fruit_poetry/pineapple.txt
108 fruit_poetry/watermelon.txt`}
</CommandOutput>
</CommandSession>

<Gloss>Count the number of lines per matching file using <kbd>wc</kbd>.
</Gloss>

## Find only files

<CommandSession>
<Command>find . <Flag kw={1}>-type f</Flag> -name 'papaya*'</Command>
<CommandOutput>{`./pictures/papaya.png
./fruit_poetry/papaya.txt`}
</CommandOutput>
</CommandSession>

## Find directories

<CommandSession>
<Command>find . <Flag kw={1}>-type d</Flag> -name 'papaya*'</Command>
<CommandOutput>./games/papaya_simulator
</CommandOutput>
</CommandSession>
